¿Qué es el motherboard?

Circuito impreso (PCB) que actúa como factor común donde se interconectan CPU, RAM, almacenamiento y periféricos.

  • Hardware crítico – sin él la PC no arranca.
  • Define el techo de actualización – zócalo, RAM, buses.
  • Regula voltajes mediante VRM y capacitores.

📌 Dato clave

El motherboard es el "chasis" de la computadora: soporte mecánico y eléctrico de todo el sistema.

Partes del motherboard

🖼️ partes-motherboard.jpg (esquema)
Componentes principales en placa madre

🧠 Zócalo CPU

Socket específico (LGA/PGA). Incompatible entre marcas.

📦 Ranuras RAM

DDR, DDR2, DDR3... distinto voltaje y pines.

🎮 Slot video

PCIe x16 (actual) / AGP (obsoleto).

🧩 Chipset

Puente norte (rápido) + puente sur (E/S).

🔌 Slots expansión

PCI, PCIe x1 para tarjetas adicionales.

⚡ Alimentación

ATX 24 pines + conector auxiliar CPU.

💾 Conectores

SATA (7 pines), IDE (40 pines, legacy).

🖥️ Panel frontal

Pines Power, Reset, LEDs.

🔊 Panel trasero

USB, video, audio, red, PS/2.

Dispositivos integrados

Componentes soldados al motherboard: video, sonido, red, controladores USB.

🎥 Video integrado

Rendimiento suficiente para ofimática y multimedia.

🎧 Sonido integrado

HD Audio, 7.1 canales.

🌐 Red (LAN)

Ethernet 10/100/1000 integrada.

🖼️ panel-trasero.jpg
Panel trasero típico de PC: PS/2, USB, VGA/HDMI, audio, LAN.

🔌 Panel trasero de una PC de escritorio: conectores esenciales integrados. Funciones avanzadas requieren tarjetas de expansión.

Buses del motherboard

📨 Datos

Transporta la información. Ancho en bits define throughput.

📍 Direcciones

Indica ubicación de memoria o periférico.

🎛️ Control

Señales de lectura/escritura, interrupciones.

🔹 Front Side Bus (FSB) conectaba CPU con northbridge. Hoy reemplazado por enlaces punto a punto.

Zócalo, video y RAM

ComponenteCaracterística
Zócalo CPUSocket específico (LGA775, AM2). Incompatible entre marcas.
Slot para videoPCIe x16 (actual) / AGP (obsoleto).
Ranura RAMDDR (184 pin, 2.5V), DDR2 (240 pin, 1.8V), DDR3 (240 pin, 1.5V).

Chipset: Puente Norte / Sur

  • North Puente norte: CPU, RAM, PCIe x16.
  • South Puente sur: SATA, USB, audio, red, PCI.

Arquitecturas modernas integran controlador de memoria en CPU.

Drivers (controladores)

🔌 Pequeños programas que permiten al SO reconocer y utilizar los dispositivos del motherboard.

Sin drivers, el hardware no funciona correctamente.

ROM BIOS

Read Only Memory + Basic Input Output System.

  • Memoria no volátil que almacena firmware.
  • Rutina de arranque (POST) y configuración (SETUP).
  • Batería CMOS mantiene los datos.

Chip de BIOS (ROM) con batería

Conclusión

✓ El motherboard es el núcleo estructural.

  • Interconecta todo el hardware crítico.
  • Chipset y buses gestionan comunicación.
  • Drivers y BIOS esenciales para funcionamiento.
